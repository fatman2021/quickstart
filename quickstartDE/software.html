Installieren von Programmen


[Klicke auf Bilder zum Vergrößern]

In der Regel benötigst Du eine Internetverbindung um Programme unter Bodhi Linux zu installieren.

Folge diesem Link wenn dies noch nicht der Fall sein sollte.

Das System auf dem neuesten Stand bringen

Ein System Upgrade bringt alle installierten Programme auf den neuesten Stand. Es gibt mehrere Möglickeiten dies zu erreichen aber wir besprechen hier nur die einfachste.

Im Terminal das man unter Hauptmenü>Anwendungen>Zubehör (Main Menu>Applications>Accessories) findet kopiert man dazu folgenden Code:

sudo apt-get update && sudo apt-get -y dist-upgrade

Notiz: Dieser Befehl nutzt apt-get und kann jederzeit wieder verwendet werden wenn Du das System auf den neuesten Stand bringen willst.

Bodhi Linux AppCenter

Das Bodhi Linux Team hat mit dem AppCenter eine einfache und schnelle Möglichkeit geschaffen häufig verwendete Programme zu installieren. Man findet hier keine komplette Liste aller verfügbaren Programme die nan unter Bodhi Linux nutzen kann, sondern eine nach Kategorien geordnete Sammlung die wir als „die besten für eine Aufgabe“ ansehen.

Außerdem stellt das Bodhi Linux AppCenter auch komplette Sätze von Programmen zur verfügung die ein ganzes Nutzungsszenario abdecken können. Lies dir die Beschreibungen durch und finde heraus was sie zu bieten haben!

Hier ist eine schnelle Anleitung die zeigt wie einfach man mit dem AppCenter Programme installieren kann:

  • Finde dein Programm.

    Wenn Du bereits weißt wie das Programm heißt das Du installieren willst kannst Du den Namen einfach in die Suchleiste oben rechts eingeben. .
    Falls nicht, navigier einfach durch die Kategorien der Hauptseite

  • Gehe zur Seite.

    Beide Suchmethoden liefern dir einen oder mehere Links.
    Jedes Programm verfügt über seine eigene Seite mit Beschreibung und Bildschirmfoto.

  • Installiere es!

    Es gibt drei Methoden der Installation zur Auswahl:

    • Install Now

      Die am häufigsten benutzte Installationsmethode im Bodhi Linux AppCenter. Klicke einfach auf den „Install Now“ Knopf und gib dein Passwort ein. Fertig! Das Programm wird nun automatisch heruntergeladen und installiert und kann gestartet werden sobald sich das Installationsfenster geschlossen hat.

    • Download

      Dieser Schaltknopf erlaubt dir das Herunterladen des Programmes sowie all seine Abhängigkeiten in einer einzigen Datei. Viele Nutzer finden dies hilfreich um z.B. von einem Computer mit schneller Internetanbindung Programme für einen Computer mit langsamer oder fehlender Anbindung zu Laden.

      Anweisungen wie man diese Dateien dann benutzt werden hier nicht gegeben, sind aber im Bodhi Linux AppCenter. nachschlagbar.

    • Quick Install

      Diese Methode ist etwas schneller als das normale „Install Now“ sollte aber nur benutzt werden wenn zuvor die normale Methode oder eine Aktualisierung der Paketliste durchgeführt wurde.

Synaptic Paketverwaltung

Die Synaptic Paketverwaltung, verbreitet bei vielen Debian GNU/Linux Basierten Distributionen ist bei Bodhi Linux Standardmässig installiert. Man findet sie im Hauptmenü unter Programme → Einstellungen → Synaptic (Applications → Preferences → Synaptic Package Manager).

Mit Synaptic, hast Du zugriff auf alle Pakete in den Bodhi Linux und Ubuntu 12.04 Repositories. (Eine Repositorie, auch Repo genannt, ist ein Speicher im Internet in dem Linux Distributionen ihre Programmpakete für dich zum installieren bereitstellen)

Bitte lies dir das Bodhi Linux Doc Wiki durch um eine komplette übersicht darüber zu bekommen wie man Synaptic verwendet.

apt-get & dpkg (Fortgeschritten)

Die Kommandozeilenprogramme apt-get und dpkg sind die zugrundeliegenden Programme für die oben beschriebenen Installationsmethoden. Fortgeschrittene Benutzer können hier gerne einen Blick drauf werfen, im Doc Wiki findet man die passende Anleitung dazu.

Für eine sofort Anleitung kannst Du im Terminal
man apt-get
oder
man dpkg
eingeben.

Außerdem ist die Installation aus dem Quellcode für fortgeschrittene Nutzer auch leicht zugänglich.

 

Beispiel: Installieren von Sprachpaketen

Notiz: Dieses Beispiel zeigt nur das installieren der Sprachpakete. Wie Du das System auch auf die gerade Installierte Sprache umstellst zeigt eine eigene Seite.

Um das Paket mit dem Bodhi Linux AppCenter zu installieren wählst Du einfach Hauptmenü>Bodhi Linux>Add Software. Dies wird den Standartbrowser (Midori wenn Du es nicht geändert hast) mit der Addresse unseres Appcenters öffnen. Gehe runter bis zur Kategorie "Extra" und klicke auf den Eintrag Language Packs. Nun siehst Du eine Liste der Sprachen die wir momentan zur verfügung stellen. Wähle hier die Gewünschte Sprache aus ("Language Pack DE" in deinem Fall) und installiere sie mithilfe der oben erklärten Methoden Install Now, Download, oder Quick Install.



Um das Sprachpaket mithilfe des Synaptic Paket Managers zu installieren startest Du Hauptmenü>Anwendungen>Einstellungen>Synaptic Paket Manager (Main Menu>Applications>Preferences>Synaptic Package Manager). Danach musste Du dein Password eingeben und bestätigen. Bei der ersten Benutzung von Synaptic solltest Du den Reload Knopf oben links in der Werkzeugleiste drücken, dies sorgt dafür das die angezeigte Paketliste auch auf dem neuesten Stand ist. Nun kannst Du mithilfe des Suchfeldes das sich ebenfalls in dieser Werkzeugleiste befindet nach dem gewünschten Sprachpaket suchen, tippe dazu "bodhi language" ein und drücke die Eingabetaste. In der nun gezeigten Liste wählst Du den Eintrag der mit "de" endet, mache einen Rechtsklick in die Zeile oder einen Linksklick in das Kontrollkästchen und wähle Mark for Installation im danach erscheinenden Menü. Danach zeigt dir das Programm alle zusätzlichen Pakete an die benötigt werden damit deine Auswahl installiert werden kann, bestätige dies mit dem Mark Schaltknopf. Nun musst Du das Ganze nurnoch mit dem Apply Knopf in der Werkzeugleiste bestätigen, hier wird dir dann nochmal eine Zusammenfassung von dem gezeigt was heruntergeladen und installiert werden muss. Auch hier immer schön Apply klicken damit die Installation startet.



Um das Sprachpaket mithilfe von apt-get zu installieren musst Du den genauen Namen kennen. Also werden wir erstmal zeigen wie Du ihn herausfinden kannst und dann erst zur eigentlichen Installation kommen:

Ein Paket Suchen:

  1. Öffne LXTerminal, Bodhis Standard Terminal, unter Hauptmenü>Anwendugen>Zubehör(Main Menu>Applications>Accessories).

  2. Benutze diesen Befehl: apt-cache search "[Suchwort]".

    In diesem Fall sollte "bodhi language" das Suchwort sein.

    Also: apt-cache search "bodhi language".

  3. Die Liste der Sprachpakete sollte nun erscheinen. Lass das Fenster geöffnet!

Ein Paket installieren:

  1. In LXTerminal benutzt Du nun diese Befehle: sudo apt-get install -y [Paketname].

    Für uns wäre das nun das Paket "bodhi-language-de". Das zusätzliche "-y" sorgt übrigens dafür das keine "Sind Sie wirklich sicher?" Frage gestellt wird.

    Also: sudo apt-get install -y bodhi-language-de.

  2. Nachdem Du die Eingabetaste gedrückt hast wird Dir ein Schwall von Text angezeigt der angibt was nun passiert. Das muss man allerdingd nicht alles verstehen.

  3. Sobald die letzte Zeile wieder "Nutzername@Rechner" anzeigt ist der Prozess beendet.
