Verbindung zum Internet aufbauen


[click on any image to enlarge]

Tip: Wenn Du gerade eine LiveCD oder einen Live USB Stick benutzt schau hier wie man das Fehlende Icon Anschaltet.



Bodhi vereinfacht das Verbinden zum Internet oder einem Lokalen Netzwerk mit Hilfe der Netzwerkverwaltung (auch nm-applet genannt).


Im Benachrichtigungsfeld(systray) sollte sich dazu ein Symbol befinden das ähnlich wie dieses aussieht.

* Wenn Du das Benachrichtigungsfeld nicht finden kannst dann lies bitte die **Notiz am Ende der Seite


Mit einem Klick mit der Linken Maustaste öffnet sich eine Liste mit den möglichen Verbindungen. Die Netzwerkverwaltung ist in der Lage alle möglichen Verbindungen wie W-Lan, UMTS, DSl oder VPN zu verwalten. Viele Verbindungen werden automatisch eingerichtet, andere benötigen die eingabe von Passwörtern oder wie im Falle einer UMTS-Verbindung den Namen des Netzbetreibers.

Die Verbindung wird nach einer kurzen Zeit aufgebaut, beachte das sich das Symbol Verändert.

Mit Hilfe der rechten Maustaste lässt sich die Verbindungsverwaltung öffnen, hier kann jede Verbindung manuell angepasst werden. Eine häufig benutzte Option ist zum Beispiel der automatische Verbindungsaufbau der dafür sorgt das sich der Rechner mit dem Internet verbindet sobald es ihm möglich ist.

Fehler beim Verbindungsaufbau können schwer zu diagnostizieren sein, besonders auf einem Rechner der überhaupt keine Verbindung aufbauen kann. Den Besten Rat den wir Dir in diesem Fall geben können ist mithilfe einer anderen Verbindung (z.B. Kabel) oder eines anderen Systems dessen Internet funktioniert mit uns im IRC oder im Forum in Kontakt zu treten. Siehe Links


Für die Nutzer einer LiveCD oder eines Live USB Sticks:


Die Netzwerkverwaltung startet auf der Live CD nicht automatisch, erst nach der Installation wird sie von alleine gestartet. Manuell kann man sie unter Hauptmenü>Anwendungen>Internet mit dem Eintrag Netzwerk starten. Das oben Gezeigte Symbol sollte nun im Benachrichtigungsfeld zu sehen sein.

*Notiz: Bitte beachte das nicht alle Profile das Benachrichtigungsfeld anzeigen. Du kannst jederzeit ein anderes Profil unter Hauptmenü>Einstellungen>Alle>Einstellungen>Profile wählen. Ein Klick mit der Linken Maustaste auf die Arbeitsfläche öffnet das Hauptmenü. Wir empfehlen das Laptop oder Desktop Profil, beide haben das Benachrichtigungsfeld deutlich sichtbar auf einer Ablage plaziert. Natürlich kann man das Benachrichtigungsfeld zu jedem Profil hinzufügen aber dies ist nicht Thema dieser Seite.
