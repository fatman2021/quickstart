Programari habitual


*** Necessita connexió a Internet per instal·lar aquests extres. Si encara no disposa de connexió, si us plau, visiti la pàgina Connecti a Internet.

En aquesta pàgina trobarà els enllaços del Bodhi AppCenter amb paquets que no estan instal·lats per defecte però que els usuaris poden necessitar.


Multimèdia

Adobe Flash
Per poder veure contingut Flash (ex: videos YouTube). 32bit Bodhi Package / 64bit Bodhi Package


Còdecs multimèdia
Dóna suport a diferents formats mèdia que no són lliures o de codi obert (no-FOSS). 32bit Bodhi Package / 64bit Bodhi Package


Connector Java (plug-in)
Per poder veure applets Java. 32bit Bodhi Package / 64bit Bodhi Package


DVD Playback
Per permetre a les aplicacions accedir a característiques avançades del format DVD. 32bit Bodhi Package / 64bit Bodhi Package


Connectivitat

Samba
Per compartir fitxers amb PC Windows de la xarxa. 32bit Bodhi Package / 64bit Bodhi Package


Bluetooth
Connexió a altres dispositius via Bluetooth. 32bit Bodhi Package / 64bit Bodhi Package



Administració

Eines de sistema
Diferents aplicacions GUI per administrar el sistema Bodhi AppCenter System Tools Directory


Instal·lador NVIDIA
Una eina per localitzar i instal·lar drivers de vídeo per a targetes de vídeo NVIDIA 32bit Bodhi Package / 64bit Bodhi Package
