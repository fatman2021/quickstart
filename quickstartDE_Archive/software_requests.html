Nützliche Erweiterungen


*** Um diese Extraprogramme zu installieren wird eine bestehende Internetverbindung benötigt. Solltest du noch nicht verbunden sein besuche die verbinden Seite

Hier sind die AppCenter Links zu Packeten die nicht automatisch installiert aber häufig erfragt werden:


Multimedia

Adobe Flash
Um Flash basierte Webinhalte (z.B.: YouTube Videos) zu betrachten. 32bit Bodhi Package / 64bit Bodhi Package


Multimedia Codecs
Erlauben das abspielen von Lizensierten Video und Tonformaten . 32bit Bodhi Package / 64bit Bodhi Package


Java Erweiterung
Um Java basierte Webapplikationen zu sehen oder Java Programme zu starten. (es existiert auch eine nicht kommerzielle Version) 32bit Bodhi Package / 64bit Bodhi Package


DVD
Damit Anwendungen DVD eigene Funktionen (Menüs,Sprachauswahl) nutzen können. 32bit Bodhi Package / 64bit Bodhi Package


Netzwerk

Samba
Tausche Dateien mit Windows PCs im Lan. 32bit Bodhi Package / 64bit Bodhi Package


Bluetooth
Bluetooth Verbindungsverwaltung. 32bit Bodhi Package / 64bit Bodhi Package



Computerverwaltung

Systemprogramme
Eine Liste mit Grafischen Programmen zu Systemverwaltung Bodhi AppCenter System Tools Directory


NVIDIA Installer
Ein Werkzeug das den Nvidia Treiber von der Herrstellerseite herunterlädt und installiert. (Möglicherweise beschädigt im Moment) 32bit Bodhi Package / 64bit Bodhi Package
