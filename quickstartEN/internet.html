Connecting to the Internet


[click on any image to enlarge]


Bodhi 3.0 brings a very visible change to it's network management tools. Instead of the Network Manager, Bodhi now comes with ConnMan. While the underlying functionality of ConnMan is identical to Network Manager, the User Interface is quite different.
Any wired connections will be turned on by default and be indicated by the icon above. Wifi, however, is turned off by default.


Each of the Widget icons that you will see in the iBar at the top of your default screen will have a status indicator.
  • Green means the connection is good and a network address was assigned to the network interface.
  • Orange means the connectivity is somehow decreased, for example the network address has not been assigned yet.
  • Blinking Orange means it is attempting to connect.


To turn on WiFi, left-click the widget in the ibar and check the WiFi On box. Then choose the type of connection from the available list. Connection Manager is capable of managing many types of network connections, including Ethernet, mobile broadband, wireless, VPN and DSL. Many connections will be configured automatically; some will need input from the user, such as a password.

After a few moments, a connection should be established. (Noted by the Green indicator)

Connections can be edited by left-clicking on the icon, clicking the 'Configure' button which opens the EConnMan window. Once a connection is established the icon will change to signal the news. A common feature that is enabled automatically is to have the connection in question automatically connect as soon as it is detected.

Connection problems can be notoriously difficult to diagnose, especially on a system that is not able to make a connection at all. If these steps fail, the best advice we can give is to get on a system that does make a connection (or use a wired connection if wireless is failing) and join us on IRC or the Bodhi Forums.
