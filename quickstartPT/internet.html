Conectando à Internet


[clique em qualquer imagem para aumentar]

Nota: Se está a usar um liveUSB/CD, clique aqui para um breve "how-to" para habilitar o nm-applet em falta.



O Bodhi Linux utiliza o Network Manager (aka nm-applet) para efectuar a ligação à Internet ou área local de um modo fácil e intuitivo.


Você verá um ícone semelhante a este nos indicadores do Sistema (systray).

* Se não vê um indicador do sistema algures na sua área de trabalho, por favor leia a **Nota na secção LiveUSB/CD no final da página.


Tudo o que tem a fazer é clicar nele com o botão esquerdo do rato. Em seguida escolha o tipo de ligação das disponíveis da lista. O Network Manager é capaz de gerir vários tipos de ligação à rede, incluindo ethernet, banda larga móvel, sem fios, VPN e DSL. Muitas das ligações são configuradas automaticamente; algumas necessitam de algum tipo de introdução de dados como a palavra passe.

Após breves momentos a conexão selecionada será estabelecida.

As ligações podem ser editadas clicando com o botão direito do rato sobre o ícone. Veja como o ícone tem agora uma aparência diferente, indicando que existe uma conexão activa como no screenshot à direita. Uma funcionalidade comum que está habilitada automaticamente é ter a ligação em questão automaticamente efectuada assim que seja detectada.

Os problemas de conexão podem ser muito difíceis de diagnosticar, especialmente num sistema que não efectua nenhum tipo de ligação. Se estes passos falharem, o melhor conselho que podemos dar é usar um sistema capaz de se conectar (ou use uma ligação cableada se a ligação sem fios está a falhar) e junte-se a nós no IRC ou nos Fórums Bodhi. Veja a nossa página de Recursos.


Para aqueles usando um LiveUSB/CD...


Como o Network Manager (nm-applet) é uma aplicação de início automatico e o Bodhi LiveUSB/CDs não a suporta actualmente, o Network Manager não se inicia por defeito. Para o iniciar manualmente vá a Principal>Aplicações>Internet e clique em Network. Deve ver um ícone semelhante ao do início da página nos indicadores do Sistema, podendo agora seguir as instruções deste guia para efectuar uma conexão a uma rede.


*Nota: Tenha em atenção que após a instalação do Bodhi Linux, o Network Manager inicia automaticamente pois este é o modo de funcionamento por defeito.

**Nota: Tenha em atenção, também, que alguns Perfis não têem os indicadores de Sistema carregados por defeito na área de trabalho. Pode escolher um Perfil diferente em Principal>Definições>Tudo>Definições>Perfis. Um clique com o botão esquerdo do rato na área de trabalho abre o menu Principal. Recomendamos a utilização do Perfil Laptop ou Desktop, porque ambos têem os indicadores de Sistema bem visíveis num Painel. Também pode adicionar manualmente o Gadget systray a qualquer Perfil, mas essa operação está fora do contexto da secção deste Guia.
